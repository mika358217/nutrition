# Дневник питания

PWA-приложение для трекинга КБЖУ.

## Файлы

- `index.html` — основное приложение
- `data.json` — база продуктов и рецептов
- `manifest.json` — PWA манифест

## Деплой на GitHub Pages

1. Создай новый репозиторий на github.com (например `nutrition`)
2. Загрузи все три файла (`index.html`, `data.json`, `manifest.json`)
3. Зайди в Settings → Pages → Source: Deploy from branch → main → / (root)
4. Через ~1 минуту приложение будет доступно по адресу `https://username.github.io/nutrition`
5. Открой на телефоне → нажми «Поделиться» → «На экран домой» — готово

## Как добавить продукт

- Через кнопку «+» в разделе «База» — введи вручную
- Через штрих-код (в следующей версии)

## Данные

Все данные хранятся локально в браузере (IndexedDB/localStorage).
Экспорт дневника: Настройки → «Скачать дневник за сегодня»
